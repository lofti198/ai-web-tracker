# Инструкции для Claude Code в этом репозитории

Полные инструкции (как добавлять парсеры, чем пользоваться для
исследования целевого сайта, что можно/нельзя менять) — в [AGENTS.md](AGENTS.md).
Этот файл читают и другие агенты (Codex и т.д.), поэтому он вынесен
отдельно, а не продублирован здесь.

@AGENTS.md

Коротко: перед тем как писать новый парсер под сайт — используй
браузерные инструменты (`claude-in-chrome` / `computer-use` MCP), чтобы
реально посмотреть сеть/DOM сайта, а не угадывать структуру данных.

**При любом изменении кода бампай `"version"` в `manifest.json`** (patch,
+1). Без исключений — это единственный признак в `chrome://extensions`,
что новая версия реально подхватилась.
